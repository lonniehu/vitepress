
# 防抖与节流

```vue
<template>
  <div>
    <p>说明：鼠标在以下元素不断移动，将会不断执行一个数值累加事件，但中间分别加入了防抖和节流函数。</p>

    <h2>防抖</h2>
    <p>在鼠标停止移动后300ms执行一次数值累加事件。</p>
    <div id="content" @mousemove="debounce(count,300)()">{{ count1 }}</div>
    <h2>节流</h2>
    <p>在鼠标移动过程中，每300ms执行一次数值累加事件。</p>
    <div id="content2" @mousemove="throttle(throttleCount,1000)()">{{ count2 }}</div>
  </div>
</template>

<script>
export default {
  data() {
    return {
      count1: 0,
      count2: 0,
      timer: null,
      timer2: null,
    }
  },
  methods: {
    debounce(fn, delay = 200) {
      console.log('debounce')
      return (...args) => {
        clearTimeout(this.timer)
        return new Promise((resolve) => {
          this.timer = setTimeout(() => {
            resolve(fn(...args))
          }, delay)
        })
      }
    },
    throttle(func, wait) {
      console.log('throttle')
      return (...args) => {
        const context = this;
        if (!this.timer2) {
          this.timer2 = setTimeout(() => {
            func.apply(context, ...args)
            this.timer2 = null;
          }, wait)
        }
      }
    },
    count() {
      console.log('count')
      this.count1++
    },
    throttleCount() {
      console.log('throttlecount')
      this.count2++
    }
  }

}
</script>

<style>

#content2 {
  height: 150px;

  line-height: 150px;
  text-align: center;
  background-color: #ccc;}
h2 {margin: 10px 0;}

p {margin: 0;
  color: #666;}
</style>
```
